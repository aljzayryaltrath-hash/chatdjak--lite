# 🌐 djakchat Lite — شبكة اجتماعية على الإنترنت

## كيفية الرفع على Railway (مجاناً) في 5 دقائق

### الخطوة 1 — GitHub
1. افتح **github.com** وسجّل دخول بحساب Google
2. اضغط **New repository** → اسمه `djakchat`
3. ارفع هذه الملفات كلها

### الخطوة 2 — Railway
1. افتح **railway.app**
2. اضغط **Login with GitHub**
3. اضغط **New Project** → **Deploy from GitHub repo**
4. اختر `djakchat`
5. انتظر دقيقتين ✅

### الخطوة 3 — الرابط
1. في Railway اضغط على المشروع
2. اضغط **Settings** → **Generate Domain**
3. ستحصل على رابط مثل: `djakchat-production.up.railway.app`
4. **أرسله لأصدقائك!** 🎉

## الملفات
- `app.py` — السيرفر
- `static/index.html` — الواجهة
- `requirements.txt` — المكتبات
- `Procfile` — أمر التشغيل
- `railway.toml` — إعدادات Railway
